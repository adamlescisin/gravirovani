# Gravírování Špytko — Web prototyp

Interaktivní HTML prototyp nového webu pro **gravirovani.com**.

## Soubory

| Soubor | Popis |
|--------|-------|
| `index.html` | Kompletní jednostránkový web (produkční verze) |
| `navrh_webu_klient.html` | Verze s prototypovým bannerem pro odeslání klientovi |
| `logo.svg` | Vektorové logo firmy |
| `analyza_trhu_seo_strategie.docx` | Analýza trhu a SEO strategie |

## Technologie

- Čisté HTML5 / CSS3 / Vanilla JavaScript
- Žádné externí závislosti — 100% offline funkční
- Responzivní design (mobile-first)
- Inline SVG grafika, systémové fonty

## Sekce webu

1. **Hero** — Headline, CTA, statistiky
2. **Služby** — 6 karet (cedule, štítky, jmenovky, dárky, reklamní předměty, zakázky)
3. **Proč my** — 4 pilíře
4. **Materiály** — 8 typů materiálů
5. **Portfolio** — Ukázky práce
6. **Kalkulátor ceny** — Interaktivní orientační výpočet
7. **Poptávkový formulář** — Formulář s GDPR souhlasem
8. **Kontakt**

## Nasazení

Web je připraven k přímému nasazení — stačí zkopírovat `index.html` na hosting nebo aktivovat **GitHub Pages** (Settings → Pages → Branch: main → Save).
